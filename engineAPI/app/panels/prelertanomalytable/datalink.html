Time stamped indices will use the time range of your selected anomaly to create a list of indices that match a specified timestamp pattern. This can be very efficient for some data sets (eg, logs) For example, to match the default logstash index pattern you would use [logstash-]YYYY.MM.DD. The [] in "[logstash-]" are important as they instruct Kibana not to treat those letters as a pattern. You may also specify multiple indices by separating them with a comma(,). For example [web-]YYYY.MM.DD,[mail-]YYYY.MM.DD

See http://momentjs.com/docs/#/displaying/format/ for documentation on date formatting.
